# Findings and Recommendations

This section summarizes the key security gaps identified during the internal audit and provides practical recommendations to reduce risk and improve compliance at Botium Toys.

## Key Findings
- Excessive user access to sensitive data (PII, SPII, payment data)
- Lack of data encryption
- No IDS/IPS or centralized security monitoring
- No backup or disaster recovery plan
- Weak password management and access control
- Use of legacy systems without proper risk management
- Partial compliance with GDPR and non-compliance with PCI DSS

## Recommendations
- Enforce least privilege access and separation of duties
- Encrypt sensitive data at rest and in transit
- Implement IDS/IPS and continuous security monitoring
- Establish regular backups and a disaster recovery plan
- Use a centralized password manager and enforce strong password policies
- Upgrade, isolate, or decommission legacy systems
- Align controls with PCI DSS and GDPR requirements
